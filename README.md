SaaS Subscription Churn Analytics Dashboard

📌 Overview

This project is an interactive Power BI dashboard developed to analyze SaaS subscription performance. It provides insights into customer subscriptions, monthly recurring revenue (MRR), customer satisfaction, churn reasons, subscription plans, and product feature usage, enabling data-driven business decisions.

🎯 Features

Interactive KPI Cards

Customer Churn Analysis

Revenue by Subscription Plan

Monthly Customer Signup Trends

Subscription Plan Distribution

Top 10 Most Used Features

Interactive Filters (Country, Industry, Subscription Plan)

🛠️ Tools Used

Microsoft Power BI

Power Query

DAX (Data Analysis Expressions)

📊 Key Metrics

Total Accounts

Total Monthly Recurring Revenue (MRR)

Churn Events

Average Customer Satisfaction

📂 Dashboard Insights

Analyze customer churn reasons.

Compare revenue across subscription plans.

Monitor monthly customer signups.

Track customer satisfaction.

Understand product feature usage.
